1 프로젝트 개요

서비스명
운명 API (Unmyeong API)
샤머니즘 총집합 운세 서비스
한줄 소개
원스톱 종합 운세 분석
생년월일시 + 혈액형 → AI 종합 해석 REST API
타겟 사용자
20~30대 MZ세대
운세/성격 분석에 관심 있는 사용자 & 개발자

배경 및 목적

MZ세대는 MBTI, 별자리, 사주, 혈액형 등 성격/운세 콘텐츠를 가볍게 소비하지만, 각각 따로 찾아봐야 하는 불편함이 있습니다.

개별 결과가 아닌, AI가 모든 결과를 종합해서 하나의 스토리로 풀어주는 원스톱 종합 분석 서비스를 API 형태로 제공합니다.

유저 입력 (최소화)

필드필수비고
생년월일필수양력 기준
태어난 시간필수모르면 시주 추정 퀴즈로 대체
혈액형선택모르면 해당 모듈 스킵

2 분석 모듈

☸
모듈 1: 사주팔자
Four Pillars of Destiny
입력생년월일시 (양력)
출력년주 / 월주 / 일주 / 시주, 천간 · 지지, 띠
로직절기 기반 간지 계산
☥
모듈 2: 오행 분석
Five Elements Analysis
입력사주팔자 결과
출력목/화/토/금/수 분포, 본인 오행, 강약 분석
☆
모듈 3: 별자리
Western Zodiac
입력생월일
출력별자리, 원소(불/물/흙/바람), 수호성, 성격 키워드
로직양력 날짜 기반 12별자리 매핑
💩
모듈 4: 혈액형 성격
Blood Type Personality
입력혈액형 (A / B / O / AB)
출력성격 특징, 장점, 단점, 연애 스타일
로직혈액형별 성격 상수 데이터
🧠
모듈 5: MBTI 예측
MBTI Prediction (엔터테인먼트)
입력사주 + 별자리 + 혈액형 종합 결과
출력예상 MBTI 타입, 각 축별 경향 점수
로직오행 · 별자리 원소 · 혈액형 성격의 상관 매핑
참고과학적 근거 X, 엔터테인먼트 목적 명시
🐒
모듈 6: 띠 운세
Chinese Zodiac
입력생년
출력띠 동물, 성격, 올해 운세, 궁합 띠
로직12간지 동물 매핑 + 상수 데이터
❔
모듈 7: 시주 추정 퀴즈
Birth Hour Estimator
입력간단한 질문 4개에 대한 답변
출력추정 태어난 시간대 (상위 2~3개 후보)
로직전통 명리학 기반 성향-시주 매핑 + AI 보정

퀴즈 질문

#질문선택지
1형제/자매 중 몇째인가요?외동 / 첫째 / 중간 / 막내
2아침형인가요, 저녁형인가요?아침형 / 저녁형 / 둘 다 아님
3첫인상이 어떻다는 말을 자주 듣나요?차가움 / 따뜻함 / 활발함 / 조용함
4의사결정 스타일은?즉흥적 / 신중함 / 상황에 따라 다름
🤖
모듈 8: AI 종합 해석
Claude AI Synthesis
입력모든 모듈의 분석 결과
출력종합 성격, 올해 운세, 연애운, 직업운, 조언
AIClaude API (Anthropic)
톤친근하고 재밌는 말투, 긍정적 방향 제시

3 API 설계

엔드포인트

GET /health 서버 상태 확인 인증 불필요
POST /api/analyze 전체 모듈 분석 (AI 해석 제외) 인증 불필요
POST /api/analyze/full 전체 분석 + AI 종합 해석 API Key
POST /api/saju 사주팔자만 분석 인증 불필요
POST /api/zodiac 별자리만 분석 인증 불필요
POST /api/blood-type 혈액형 성격만 분석 인증 불필요
POST /api/estimate-hour 시주 추정 퀴즈 인증 불필요
POST /api/compatibility 두 사람 궁합 분석 API Key

7 MVP 범위

In Scope (v1.0)

  • ☐ 사주팔자 계산 (절기 기반)
  • ☐ 오행 분포 분석
  • ☐ 별자리 분석
  • ☐ 혈액형 성격 분석
  • ☐ 띠 운세
  • ☐ MBTI 예측 (재미 요소)
  • ☐ 시주 추정 퀴즈
  • ☐ Claude API 종합 해석
  • ☐ 두 사람 궁합 분석
  • ☐ Swagger API 문서

Out of Scope (v2.0 고려)

  • ☐ 음력 입력 지원
  • ☐ 일별/월별 운세 (데일리 운세)
  • ☐ 타로 카드 연동
  • ☐ 관상/손금 (이미지 분석)
  • ☐ 풍수 분석
  • ☐ 사용자 계정 & 결과 저장
  • ☐ 프론트엔드 웹 UI
  • ☐ 결과 공유 카드 이미지 생성
